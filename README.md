# Scradient 🎨

[![Website](https://img.shields.io/website?url=https%3A%2F%2Fscradient.theritvik.in)](https://scradient.theritvik.in)
[![PWA](https://img.shields.io/badge/PWA-Ready-blue)](https://scradient.theritvik.in)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

A beautiful, smooth gradient screensaver that creates calming color transitions. Perfect for relaxation and ambient displays.

![Scradient Demo](https://github.com/raghav-rama/scradient/assets/91389059/d5f50171-57dc-4ff7-895b-24ca863e535a)

## ✨ Features

- 🌈 Smooth gradient transitions
- 📱 PWA support - install on any device
- 🖥️ Fullscreen mode
- 📴 Offline support
- 📱 Mobile responsive

## 🚀 Quick Start

Just visit [https://scradient.theritvik.in](https://scradient.theritvik.in) and click "Start"!

### 💻 Install as Desktop App

1. Visit the website in Chrome
2. Click the install icon in the address bar
3. Enjoy Scradient as a desktop app!

### 📱 Install on Mobile

1. Visit the website in your mobile browser
2. Tap "Add to Home Screen"
3. Launch Scradient from your home screen!

## 🛠️ Development

```bash
git clone https://github.com/raghav-rama/scradient.git
cd scradient
code .
# serve with any static server
```

## 🎮 Controls

- **Start Button**: Begin the screensaver
- **Fullscreen Button**: Toggle fullscreen mode
- **Stop Button**: Exit screensaver mode
- **ESC**: Exit fullscreen

## 🔧 Technical Details

- Pure JavaScript with no dependencies
- CSS3 animations and transitions
- Hardware-accelerated rendering
- Service Worker for offline support
- Progressive Web App (PWA)
